# 碳纤维领域知识

## 一、产业概述

碳纤维（Carbon Fiber）是一种含碳量在90%以上的高强度、高模量新型纤维材料，具有重量轻、强度高、耐腐蚀、耐高温等优异性能，被誉为"黑色黄金"。碳纤维及其复合材料是现代工业、国防军工和高新技术产业不可或缺的关键基础材料。

### 1.1 制造流程

碳纤维的制造流程主要包括以下环节：

1. **前驱体制备**：以聚丙烯腈（PAN）为主要原料，纺丝制成原丝
2. **氧化稳定化**：在200-300°C下进行预氧化，使线性分子结构环化
3. **碳化**：在1000-1500°C惰性气氛下碳化，去除非碳元素
4. **石墨化**（可选）：在2500-3000°C高温下石墨化，提升模量
5. **表面处理与上浆**：改善纤维表面性能，增强与树脂基体的界面结合力

### 1.2 主要分类

**按原料分类：**
- PAN基碳纤维（聚丙烯腈基，占全球产量90%以上）
- 沥青基碳纤维（各向同性/中间相沥青基）
- 粘胶基碳纤维（主要用于刹车片等摩擦材料）

**按性能分级：**
- 标准强度型：T300级（抗拉强度3.5GPa）
- 高强型：T700级、T800级
- 超高强型：T1000级、T1100级
- 高模型：M40级、M55级、M65级
- 高强高模型：HM系列

**按用途分类：**
- 航空航天级（T800及以上级别）
- 工业级（T700级别为主）
- 体育休闲级（T300-T700级别）

### 1.3 应用领域

- **航空航天**：飞机机身、机翼、卫星结构件、火箭燃料箱
- **汽车轻量化**：车身面板、传动轴、刹车盘
- **风电能源**：风电叶片、压力容器、储氢瓶
- **体育休闲**：高尔夫球杆、钓鱼竿、自行车架
- **建筑加固**：碳纤维布/板结构加固
- **电子电气**：散热材料、电磁屏蔽

### 1.4 产业链结构

上游：PAN原丝 → 中游：碳纤维 → 下游：碳纤维复合材料 → 终端应用

## 六、技术发展趋势

1. **高性能化**：T1000/T1100级碳纤维国产化突破
2. **低成本化**：大丝束碳纤维产业化降低成本
3. **绿色制造**：低碳、节能的碳纤维制造工艺
4. **复合材料设计**：CFRP轻量化设计优化
5. **回收再利用**：碳纤维复合材料回收技术
